# sleuthkit_test_data
Holds images and other test data for sleuthkit


# Usage
* Remember to [enable GIT LFS](https://docs.github.com/en/repositories/working-with-files/managing-large-files/configuring-git-large-file-storage)

```
git lfs track "*.dd"
git lfs track "*.E01"
git lfs track "*.iso"
```
